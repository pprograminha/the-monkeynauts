## **Run the application under development**

**Install dependencies**
```
1. yarn
```

**Run the docker image**
```
2. docker-compose up -d
```

**Run migrations**
```
3. yarn prisma migrate dev
```

**Launch application**
```
4. yarn dev:server
```

**Run all cmd**
ubuntu: sh ./scripts/init.sh 
bash: bash ./scripts/init.sh
